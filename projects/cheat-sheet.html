Ruby Cheatsheet: Hashes, Arrays, Strings and Integers

Hash 

         
Example of hashes: 
    grades = { "Jane Doe" => 10, "Jim Doe" => 6 }
    options = { :font_size => 10, :font_family => "Arial" }          
    options = { font_size: 10, font_family: "Arial" }
        
Creating new hashes:  
    grades = {}
    grades.default = 0
    grades = Hash.new
    grades = Hash.new(0)
   
Adding or modifying key/value:
    grades["Dorothy Doe"] = 9
    books[:matz]  = "The Ruby Language"
            
Accessing hashes:
    grades["Dorothy Doe"]
    books[:matz]
            
    h = { "a" => 100, "b" => 200 }
      h["a"]   # 100
      h["c"]   # nil

A hash is a dictionary-like collection of unique keys and their values. Also called associative arrays, they are similar to arrays, but where an array uses integers as its index, a hash allows you to use any object type.

Some things to know about hashes:

  • Hashes use curly brackets { }
  • We access hash values by using the keys inside [ ] brackets with quotes or colon depending on the key, see examples above
  • Keys should be unique
  • If the key doesn't exist then we get nil
  • We can set a default value with grades = {} then grades.default = 0 or grades = Hash.new(0)

Learn more about Ruby hashes and hash methods here.

Array 

          
Examples of arrays: 
    ary = [1, "two", 3.0] #=> [1, "two", 3.0]
    a = [ 1, 2, [true, [4, "hello"] ] ]
              
Creating new arrays: 
    ary = []
    ary = Array.new    #=> []
    Array.new(3, true) #=> [true, true, true]
              
Adding or modifying elements:
    ary[0] = 4
    ary[1] = "hello world"
          
Accessing arrays: 
    arr = [1, 2, 3, 4, 5, 6]
      arr[2]    #=> 3
      arr[100]  #=> nil

Arrays are ordered, integer-indexed collections of any object. Array indexing starts at 0, as in C or Java. A negative index is assumed to be relative to the end of the array---that is, an index of -1 indicates the last element of the array, -2 is the next to last element in the array, and so on.

Some things to know about arrays:

  • Arrays use [ ] brackets
  • We access array elements using indexes like the example ary[0], this means index 0 of array ary
  • Nested array means an array with sub-array elements, example [ 1, 2, [true, [4, "hello"] ] ]
  • Trying to access an array index that doesn't exist will get nil
  • By using %w{ David A. Black , we can get ["David", "A.", "Black"]
  • We can convert an array into a string by using the join method

Learn more about Ruby arrays and array methods here.

String 

          
Examples of strings: 
    my_string = "hello world"
    my_string = String.new("hello world")
    "1 2 3 4 5 6, yes this is a super long string!"
              
Creating new strings: 
    'creating a new string within single-quotes'
    "or create a new string within double-quotes"
    a = "or assign a string to a variable"
              
Adding or modifying strings:
    a = "hello "
      a << "world"   #=> "hello world"
      a.concat("world")  #=> "hello world"
          
Accessing strings:
    a = "hello there"
      a[1]   #=> "e"
      a[2, 3]   #=> "llo"
      a[2..3]   #=> "ll"

A string is a group of characters that typically make up human readable words or sentences.

Some things to know about strings:

  • Strings go within single or double quotes
  • String interpolation doesn't work with single-quoted strings
  • We can convert a string into a number or array using methods such as string.to_i and split method

Learn more about Ruby strings and string methods here.

Integer 

          
Examples of integers: 
    2016
    123456
              
Creating new integers: 
    2016
    a = 2016
              
Adding or modifying integers:
    2016 * 20 + 1
    a = 123456 * 2 - 10
    a + 100000

An integer means a whole number (no decimals).

Some things to know about integers:

  • Integers have limited methods but we can change it into a string with 12345.to_s and then use string methods
  • You can perform all the basic arthimetic calculations using -, +, * and /
  • A number with decimal is called a float

Learn more about Ruby integers and integer methods here.
