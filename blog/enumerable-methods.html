Enumerable Method: Map

March 10, 2016

The Map Method

The map method is a highly useful Ruby method. Let's look at a real life example, you own a store and you have a list of products with their prices. One day you decided to double the price of all your products...because it's your store and you just felt like it. This is where we use the map method.

So what does the map method do? Map goes through a provided object, does something to it then gives you the results. Some examples of what this looks like:

  • my_array = [2,4,6,8,100]
    my_array.map { |num| num + 2 }
    • In this example, my_array is an array (aka an object) and it contains 5 numbers: [2,4,6,8,100]
    • We use the map method on it by adding a period after my_array followed by map
    • Then we add a bracket and a vertical bar, both of which are part of the map method's syntax
    • Next we see num which is just a name I made up as a variable to represent each of the 5 numbers in my_array but you can use something else like x
    • After that we see a closing veritical bar
    • Finally we see num + 2 which means we are using each of the 5 numbers in my_array and adding 2 to them and of course we have the closing bracket
    • If we use the above codes, we'll see an output of [4, 6, 8, 10, 102]
  • Let's take a look at another example:
  • my_inventory = {
        "pizza" => 2,
        "toy car" => 3,
        "book" => 20,
        "pencil" => 1,
        "TV" => 500,
      }
      my_inventory.map { |item, price| price * 2 }
    • my_inventory is a hash (aka an object) and it contains 5 items along with their prices
    • We use the map method the same way as the above example except we we use item instead of num, we add an extra variable called price and we added a comma between them to show these are two variables
    • After that we see a closing veritical bar
    • Finally we see price * 2 which means we are using each of the items in my_inventory and we are multiplying their price with 2
    • If we use the above codes, we'll see an output of [4, 6, 40, 2, 1000]

To Map or Map!

There are 2 ways to use map, there's the map and there's map! (with exclamation). Map will output results with a new object while map! will change and output results using the existing object. The terms used to describe this is non-destructive (map) and destructive (map!).
